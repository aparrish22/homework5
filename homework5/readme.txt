
To run 'homework5' on xampp,

	Insert and extract the homework5 folder from homework5.zip into 'htdocs' within your xampp file

	In localhost/phpmyadmin/, create a database called 'user_information_hw5'.

	Start up XAMPP and run apache and mysql

	Go to a web browser and enter 'localhost/homework5'
